# Image

Use this component to display a image.

<Canvas>
  <Story name="Component">
    <DisplayImage
      alt="Image alt text"
      height={480}
      url="https://nextjs.wpengine.com/wp-content/uploads/2021/02/nextjs-wordpress-starter.jpg"
      width={640}
    />
  </Story>
</Canvas>

## Image + Link

Add a link to the image using the `href` prop.

<Canvas>
  <Story name="Image + Link">
    <DisplayImage
      alt="Image alt text"
      caption="This is the image caption"
      height={480}
      href="https://webdevstudios.com"
      url="https://nextjs.wpengine.com/wp-content/uploads/2021/02/nextjs-wordpress-starter.jpg"
      width={640}
    />
  </Story>
</Canvas>
